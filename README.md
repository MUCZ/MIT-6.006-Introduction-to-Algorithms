##### 文件：

- 本repo排序和整理了官网上提供的文档，加了一些自己的笔记和测试代码
- 文件夹前的编号是为了正确显示lec和ps的顺序，顺序参考了课程页面

- `lec` ：讲义 or ppt ，与视频正课编号一致
- `ps `：problem set 也就是题
